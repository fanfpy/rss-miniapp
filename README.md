# RSS MiniApp

RSS MiniApp 是一个简单的 RSS 订阅器，使用 Uniapp 开发，适用于微信小程序。UI 库使用了 uViewUI。

## 功能
- 添加和管理 RSS 订阅源
- 查看和阅读订阅文章
